# Application Naming Recommendations

*Research conducted: 2025-11-11*

## Executive Summary

Based on extensive research of the application's features and internet uniqueness checks, we recommend renaming "Academic Explorer" to better differentiate it in the market. The top recommendations are **CitationMesh** or **ScholarWeave**.

## Application Context

Academic Explorer is a TypeScript-based research platform that provides:
- Interactive knowledge graph visualizations of academic literature
- Force-directed graph layouts with deterministic positioning
- OpenAlex API integration for exploring scholarly publications, authors, institutions, and topics
- Progressive Web App with offline capabilities
- Multi-tier caching system (Memory → localStorage → IndexedDB → API)
- Entity-centric browsing supporting 7 entity types (Works, Authors, Sources, Institutions, Publishers, Funders, Topics)

## Final Recommendation

**Primary choice**: **ScholarWeave** or **CitationMesh**

Both names are:
- Highly unique with no existing conflicts
- Immediately convey the platform's purpose
- Memorable without being gimmicky
- Professional and appropriate for academic context
- Available for branding across multiple platforms

**ScholarWeave** edges ahead slightly for its elegant, evocative quality and perfect memorability score, while **CitationMesh** is more technically descriptive and may resonate better with researchers familiar with network terminology.

## Research Methodology

This research involved:
- Comprehensive codebase analysis to understand application features
- Web searches for 15+ name candidates
- Domain availability preliminary checks
- Conflict analysis with existing products/services
- Evaluation against uniqueness, relevance, and memorability criteria
